# Booking Management System

A full-stack appointment management system for medical or consulting service scenarios.  
It provides an admin workflow for managing people, appointments, session statuses, CSV operations, and dashboard data.

## Tech Stack

**Frontend:** React, Vite, JavaScript, HTML, CSS  
**Backend:** Node.js, Express.js, REST APIs, JWT, bcrypt  
**Database:** PostgreSQL  
**Tools/Libraries:** Luxon, Multer, csv-parse

## Key Features

- Admin login with JWT-based authentication
- People management for patients and staff
- Appointment creation, editing, cancellation, completion, and deletion
- Search, filtering, sorting, and pagination
- CSV import for people records
- CSV export for session records
- Dashboard for today’s sessions, upcoming sessions, and recent activities
- Melbourne timezone handling using Luxon
